# go-micro
## Quickstart

```bash
$ micro server

$ micro login
username: admin
password: micro

# Run the example service.
$ micro run github.com/micro/services/helloworld
$ micro status

# Run the main.go example.
$ micro run .

$ micro status

$ micro logs go-micro
Response: Hello john

# Create a new service.
$ micro new <service-name>
```
